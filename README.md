## 🇺🇸 English

> AI Agent Desktop Platform — Built on [OpenClaw](https://github.com/openclaw/openclaw), designed for everyone.

Create AI agents and deploy them to Telegram, Discord, WhatsApp, Slack, WeChat, LINE, Feishu, Signal, Microsoft Teams, Matrix, IRC, Google Chat — all from one desktop app. No coding required.

### macOS Install
1. Download `.dmg` → Open → Drag **ZoyaChat** to **Applications**
2. **Right-click** ZoyaChat.app → Click **Open** → Click **Open** in the dialog
3. Done! After first launch, double-click works normally.

> If right-click doesn't work or macOS shows a "file is damaged" warning, don't delete the app — close the dialog, open Terminal and run:
> ```
> codesign --force --deep --sign - /Applications/ZoyaChat.app && xattr -cr /Applications/ZoyaChat.app
> ```

> ℹ️ Not notarized by Apple — this is normal for independent apps, not malware.

### Windows Install
1. Download `.exe` → Double-click to run
2. Windows SmartScreen may show "Windows protected your PC" → Click **More info** → Click **Run anyway**
3. Follow the setup wizard → Launch ZoyaChat

> ℹ️ SmartScreen warning appears because the installer is not code-signed. This is normal.
